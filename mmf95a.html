Program Specifications for Program Simulations

Maarten M Fokkinga, Jaap van der Woude, Job Zwiers
We give a concise, structured, and calculational proof of a theorem that yields a program specification for a given, specific, program simulation. Keywords: program correctness, programming calculi.
