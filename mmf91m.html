Functional Programming with Bananas, Lenses, Envelopes and Barbed Wire

Erik Meijer, Maarten M Fokkinga, Ross Paterson
We develop a calculus for lazy functional programming based on recursion operators associated with data type definitions. For these operators we derive various algebraic laws that are useful in deriving and manipulating programs. We shall show that all example functions in Bird and Wadlesr's ``Introduction to Functional Programming'' can be expressed using these operators.
The full paper: postscript (21 pages).
